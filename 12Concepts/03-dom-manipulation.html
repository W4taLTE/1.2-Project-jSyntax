üìå Elements

üí° Fundamental DOM

DOM (Document Object Model) adalah representasi struktur halaman web yang memungkinkan JavaScript mengubah konten, struktur, dan style secara dinamis.

dom-manipulation.js
// Mendapatkan elemen
const header = document.getElementById('header');
const buttons = document.querySelectorAll('.btn');

// Mengubah konten
header.textContent = 'Judul Baru';
header.innerHTML = '<span>Judul</span> Baru';

// Mengubah style
header.style.color = '#9333ea';
header.style.fontSize = '2.5rem';

üìù Penjelasan Detail:

‚Ä¢ getElementById() - Memilih elemen berdasarkan ID
‚Ä¢ querySelector() - Memilih elemen pertama yang cocok dengan selector CSS
‚Ä¢ querySelectorAll() - Memilih semua elemen yang cocok
‚Ä¢ textContent - Mengubah teks dalam elemen
‚Ä¢ innerHTML - Mengubah HTML dalam elemen

dom-features.js
// Membuat elemen baru
const newDiv = document.createElement('div');
newDiv.className = 'card';
newDiv.textContent = 'Kartu Baru';

// Menambahkan ke DOM
document.body.appendChild(newDiv);

// Event listeners
buttons.forEach(btn => {
  btn.addEventListener('click', e => {
    console.log('Tombol diklik!');
  });
});

// Manipulasi class
header.classList.add('active');
header.classList.remove('hidden');
header.classList.toggle('dark-mode');

‚ÑπÔ∏è Performance Notes

Untuk manipulasi DOM yang kompleks, gunakan DocumentFragment untuk meminimalkan reflow. Event delegation lebih efisien untuk banyak elemen.

üí° Pro Tips

üéØ Best Practices

Gunakan event delegation untuk elemen dinamis, minimalkan manipulasi DOM langsung, dan cache referensi elemen yang sering digunakan.

best-practices.js
// ‚úÖ Good - Event delegation
document.addEventListener('click', e => {
  if (e.target.matches('.btn')) {
    // Handle button click
  }
});

// ‚úÖ Good - Caching elements
const header = document.getElementById('header');
const buttons = document.querySelectorAll('.btn');

// ‚ùå Avoid - Inline event handlers
<button onclick="handleClick()">Click</button>

// ‚ùå Avoid - Frequent DOM updates
for (let i = 0; i < 1000; i++) {
  document.body.appendChild(document.createElement('div'));
}

üìù Penjelasan Detail:

‚Ä¢ Event delegation - Lebih efisien untuk elemen dinamis
‚Ä¢ Cache references - Hindari query DOM berulang
‚Ä¢ Batch updates - Gunakan DocumentFragment untuk banyak perubahan
‚Ä¢ Debounce events - Untuk scroll/resize yang sering terjadi
‚Ä¢ Virtual DOM - Pertimbangkan untuk aplikasi kompleks
